### Quick Start (No Installation)

Run directly with npx:

```bash
# Run without installing
npx macx-cli-tool
```

### Installation

#### Local Development

```bash
# Clone the repository
git clone https://github.com/YourUsername/MacX.git

# Navigate to the CLI directory
cd MacX/cli

# Install dependencies
npm install

# Build the CLI
npm run build

# Start the CLI
npm start

# link
npm link
```
